Not
> Yukarıda ki yapmış olduğımız sçimler ile html sayfası üzerinde bulanan class'ı anasayfa olan bütün elemanları seçtik ve tüm ul etiketlerini seçmiş olduk bu kısım çok önemli Javascripte ki gibi ilk elemanı seçmedik.
> "a" etiketini seçmek için $("a") yazmamız yeterlidir. Fakat bu şekilde seçim yapıldığında html sayfamız üzerinde ki bütün "a" etiketlerini seçer.Ben "ul" etiketinin altında ki "li" etiketinin altında ki "a" etiketini seçmek istediğimde $(ul li a); yazmamız yereli olacakır. Bu kullanım "ul" etiketi altındaki "li" etiketeti altında ki "a" etiletini seçer. $(li a); bu şelilde yazılınca da aynı sonucu elde etmiş oluruz.
> Html sayfamızda birden fazla "ul li a" olduğu durumlarda da bütün li altında ki a etiketlerini seçer ve ben tek bir a etiketi seömek istiyor isem a etiketine id attribute eklememiz yeterli olacaktır. Not biliyorsunuz ki id attribute si benzersiz bir şekilde tanımlanmalıdır.
> $("div.box") bu kullanım şeklinde ise class'ı box olan divleri seçer. Etiket ve class aynı anda yazılarak seçim yapabiliriz.

Seçilen Elemanlara Css Özellikleri Ekleme

.css('özellik','değer);'
.css kullanarak seçilen etiketlerin css özelliğini değiştirebiliriz.
Örneğin;
$("#container").css("border","1px solid yellow"); or var style ={
border:"1px solid green",
background:"red",
color:"white",
}

$("#container").css("border","1px solid yellow"); or var style ={

background:"red",
color:"white",
}
$("#container).css(style);

.text() Metot

Seçilen elemanlar içerisinde ki textleri birleştirip döndüren metotdur.

.text() metodunun get ve set özelliği vardır.

Get metodu ile etiketinin içeriğine ulaşabilirken set sözelliği ile değiştirebiliyoruz

get metodunun kullanılışı $("li").text(); dir bu şekilde kullanıldığında html sayfamızda ki tüm "li" etiketlerinin text'tine ulaşabiliyoruz.

set metodunun kullanmı ise $("#sonuc).text("Jquery Öğreniyorum");dir bu şekilde kullanıldığında "id" si sonuç olan etikete ulaşır ve o etiketin textsini "Jquery Öğreniyorum" olarak değiştirir.


.val() Metot

Html sayfamızda bulanan "input" etiketlerinin yada "value" değeri olan etiketlerin içerisinde ki değeri alıp değiştirebildiğimiz metotdur.

.text() metodunda olduğu gibi get ve set özelliği mevcuttur

get metodu ile etiketin value değerine ulaşabilirken set metodu ile value değerini değiştirebiliriz.

Html sayfamızda bulunan bütün "input" etiketlerin value değerine "Jquery val metot" yazar
