Local & Session Storage

Yapmış olduğumuz uygulamada kullanacağımız bilgileri tarayıcı hafızasında saklama gereği duyabiliriz. Hafızaya eleman ekleme çıkartma vb. işlemler için Local & Session Storage özelliğini kullanarak saklayabiliriz.

  • Local Storage : Tarayıcı hafızasına kalıcı olarak kaydedilir.
  • Session Storage : Uygulama yada web sitesi açık kaldığı sürece hafızada kalır sayfa kapandığında bilgiler hafızada artık yer almaz.
Bir Html sayfasının hafızasına eklenen ögelere f12 veya fareden sağ tıklayıp incele dedikten sonra açılan ekranda "Application" bölümü altnda ki Local veya Session Storage bölümlerinden ulaşabilirsiniz.


Metodlar Ve Kullanım Şekilleri


> Syntax {Değişken Türü} {Değişken Adı} = localStroge.setItem('Key','Value');

Bu bölümle Local Storage üzerinden örneklendirilmişitir. Aynı örnekler kullanım şekilleri Session Storage için de geçerlidir. Tek yapılması gereken "local" yazan yerlere "session" yazmaktır

setItem(); methodu

setItem(); methodu ile tarayıcımızın hafızasına eleman ekleyebildiğimiz metoddur. 

                    
                        const isim = localStorage.setItem('İsim','Muharrem');   

                        const soyİsim = localStorage.setItem('Soyİsim','Özer');

getItem(); methodu

getItem(); methodu ise tarayıcının hafızasına eklediğimiz elemanları alabileceğimiz methoddur.
getItem(); metdhodunun kullanımı; 

                    
                        let val = localStorage.getItem('isim');

                        console.log(val);

removeItem(); methodu

removeItem(); metodu ile tarayıcı hafızasına eklediğimiz elemanları sildiğimiz metoddur.
getItem(); metodunun kullanımı; 

                    
                        localStorage.removeItem('isim'); 

                        console.log(localStorage);
Çıktı => Tarayıcının Console Bölümünde > Stroge {Soyİsim: "Özer", Length: 1} olarak görünmketedir.

clear(); metodu

clear(); metodu removeItem(); metodu gibi elemanları silen bir metoddur fakat farkı bütün hafızayı tek seferde siler. removeItem(); metodunda ise istediğimiz elemanı silebiliyor idik.
clear(); metodunun kullanımı; 

                    
                        localStorage.clear();
Çıktı => Tarayıcının Console Bölümünde > Stroge {Length: 0} olarak görünmketedir.

JSON.stringfy(); && JSON.parse(); metodları

JSON.stringfy(); Obje olarak eleman eklemek için kullanılan metoddur.
JSON.parse(); metodu ise eklenen obje elemanlara ulaşabildiğimiz metoddur.
Bir dizi tanımlayıp tarayıcı hafızasına seItem(); metodunu kullanarak ekleyebiliriz.
Fakat eklenen dizi string olarak eklenecektir.
Bu durum elemanlara ulaşmamızı zorlaştıracaktır.
Obje elemanı olarak ekleyebilmek için 

                    
                        let hobi = ['müzik','resim','seyehat']; 

                        localStorage.setItem('hobi',JSON.stringify(hobi)); 

                        

                        let hobiler = JSON.parse(localStorage.getItem('hobi')); 

                        console.log(hobiler); 

                        console.log(localStorage);
