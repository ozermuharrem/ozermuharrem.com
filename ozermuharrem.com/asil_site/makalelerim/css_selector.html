Css Selector (Seçiciler)

Css seçici, Html sayfasında bulanan element ve attribute özelliğine göre seçip stil özelliği katabilmemiz için kullanacağımız en temel konudur. Css'in neredeyse %50 lik kısmını temsil eder. Çünki seçmeden stil özelliği ekleyemeyiz.

Css seçicileri 5 kategoriye ayırabiliriz.

  • Basit Seçiciler (Elements, Id, Class'ına Göre Ögeleri Seçer)
  • Birleştirici Seçiciler (Aralarında Ki Belirli Bir İlişkiye Göre Seçer)
  • Pseudo Class Seçiciler (Sözde Sınıf Seçiciler)
  • Pseudo Elements Seçiciler (Bir Elementin Bir Bölümünü Seçer )
  • Attribute Seçiciler (Attribute Göre Seçer)

Seçici İşlemleri

  • Neyi Seçerseniz onunla ilgili işlem yaparsınız
  • CSS kodu uygulayacagız, öncelikle HTML elemanını seçmeniz gerekir.
  • işte bu aşamada selector yada seçici dediğimiz kodlar devreye girmektedir.

Secici {özellik:değer; özellik:değer}
p {color:red}
h1 {color:green; font-size:20px}


Temel Seçici Örnekleri

Seçici(CSS Kodu) Açıklama
* Herşey Seçilecek
p p tagı seçicelec
#ilk id degeri ilk olan seçilecek
.son class degeri son olan seçilecek
h1.egik class degeri egik olan h1 etiketleri seçilecek
p strong p tagı içerisinde ki tüm strong tagları seçer
nav #buton nav tagı içindeki id degeri buton olan tüm elemanlar Seçilecek
h1,h2,h3,h4,h5,h6 h1-h6 arasın da ki tüm agları seçer

Karmaşık Seçici Örnekleri

Seçici{CSS Kodu} Açıklama
.banner p class değeri banner olan eleman içerisinde ki p tagını seçer
div>p evebeyni div olan tüm tagları seçer
.menu ul li class degeri menu olan eleman içerisinde ki ul tagının içinde ki li tagını seçer
nav.menu li nav tagı içerisinde bulunan ve class degeri menu olan eleman içerisinde ki li tagını seçer
body>p:fist-child body tagı içerisinde ilk paragrafı seçer (ilk çocuk eleman)
body>p:last-child body tagı içerisinde ki son paragrafı seçer(son çocuk eleman)
:activee aktif elemanı seçer
:after sonraki elemanı seçer

Link ve Özellik Seçiciler

Seçici{CSS Kodu} Açıklama
a:link henüz ziyaret edilmemiş likleri seçer
a:visited ziyaret edilmiş linkleri seçer
a:focus klavye ile etkinleştirilmiş linkleri seçer
a:hover üzerinde fare olan linki seçer
input[type] input içinde type özelliğine sahip olanları seçer
input[type="text"] input içinde ytpe özelliği text olan olanları seçer
img[title~="kedi"] img tagında ki title özelliği kedi olanı seçer
